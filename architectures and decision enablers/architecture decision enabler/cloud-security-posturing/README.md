# Cloud Security Posturing Assessment Decision Enabler

A structured, automated Excel-based assessment calculator built to accelerate your Cloud security current posturing. 

## 🚀 Key Features

* **Cloud Security Posturing Assessment:** Evaluates current security practices and organizational alignment based on the below parameters: 

|Parameters|
   |-|
   |**IAM \& Least Privilege**|
   |**Cloud Infrastructure Perimeter Security**|
   |**Operating System Privileged Account Control**|
   |**Compute Hardening \& Vulnerability Management**|
   |**Data Protection \& Encryption at Rest/In-Transit**|
   |**Secrets \& Key Management**|
   |**Storage \& Database Security Configuration**|
   |**Backup, Disaster Recovery \& Ransomware Resilience**|
   |**Control Plane \& API Integrity**|
   |**Serverless \& Container Runtime Protection**|
   |**Audit Logging, SIEM, \& Cloud Observability**|
   |**Infrastructure as Code (IaC) \& Configuration Drift**|
   |**Cross-Account \& Third-Party Integrations**|
   |**Multi-Tenant Network Isolation**|
   |**CSPM \& Continuous Compliance Auditing**|
   |**DevSecOps \& CI/CD Pipeline Security**|   


* **AI Security Posturing Blueprint:**  Investigates current organization posturing state from below states: 

|Current State |Business Explanation|
|-|-|
|**Cloud Optimized State:**| The organization exhibits a highly mature security architecture. Technical controls are automated, segmentation is pristine, and privileged activities are stringently managed. |
|**Cloud Resilient State:**|  Core defensive perimeters are functional, and critical business validations are in place. However, some gaps exist due to manual dependencies or minor tracking lapses in asset lifecycles. | 
|**Elevated Cloud Exposure:**| Major control vulnerabilities are present. The environment lacks robust segmentation, identity management has clear gaps, and security operations are reactive. |
|**Critical Cloud Threat Posture:**|Widespread failure or complete absence of standard security baselines. Credential hygiene is absent, access groups are unmonitored, and third-party interactions introduce severe risk  |
