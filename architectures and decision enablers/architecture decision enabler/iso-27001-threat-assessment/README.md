# ISO 27001 Security Posturing Assessment Decision Enabler

A structured, automated Excel-based assessment calculator built to accelerate your  ISO 27001 Security current posturing. 

## 🚀 Key Features

* **ISO 27001 Security Posturing Assessment:** Evaluates current security practices and organizational alignment based on the below parameters: 

|Parameters|
   |-|
   |**Access Control: Identity Verification**|
   |**Operations: Endpoint Protection \& Hygiene**|
   |**System Monitoring: Logging \& SIEM**|
   |**Change Management: Software Delivery**|
   |**Change Management: Infrastructure as Code**|
   |**Network Security: Perimeter \& Segmentation**|
   |**Encryption: Data Rest \& Transit Lifecycle**|
   |**Governance: Policies \& Security Awareness**|
   |**Governance: Asset \& Inventory Tracking**|
   |**Third-Party Risk: Vendor Oversight**|
   |**Privacy \& Confidentiality Controls**|
   |**Risk Management: Vulnerability Assessment**|
   |**Data Protection: DLP \& Backup Retention**|
   |**Availability: Disaster Recovery Resilience**|
   |**Identity, Access, and Hygiene**|

* **ISO 27001 Security Posturing Blueprint:**  Investigates current organization posturing state based on below states:

|Current State |Business Explanation|
|-|-|
|**High Audit Readiness (Clean Report Forecast):**| Controls are fully designed, operating effectively, completely documented, and consistently executed across the entire organization. |
|**Moderate Readiness (Minor Exceptions):**| Core security controls are functional, but minor gaps exist in documentation, sample testing consistency, or administrative log reviews. | 
|**Low Readiness (Audit Failure / Non-Conformity Risk):**| Key security controls are missing, poorly designed, or suffer from widespread execution failures across critical trust service criteria or clauses. |
|**Unprepared (Widespread Exceptions Guaranteed):**|Widespread failure or complete absence of standard security baselines. Credential hygiene is absent, access groups are unmonitored, and third-party interactions introduce severe risk.  |
